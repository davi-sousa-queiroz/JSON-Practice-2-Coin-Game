# 💰 Coin Game (JSON Practice 2) (April 5th 2026)

This is a super simple Python game I made while learning JSON and file handling.

The goal was to understand how to save data even after the program closes — and yeah, it actually works 🔥

⸻

## 🎮 Features
	•	Add coins
	•	View your coins
	•	Data is saved using a .json file
	•	Automatically creates the file if it doesn’t exist

⸻

## 🧠 What I learned
	•	How to use json.load() and json.dump()
	•	How to create a file if it doesn’t exist (try/except)
	•	That a JSON file is basically just a Python dictionary
	•	How real games save progress

⸻

## 📁 Notes
	•	The data.json file is created automatically when you run the program
	•	Coins will keep increasing even after you close and reopen the game
